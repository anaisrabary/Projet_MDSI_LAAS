Projet de MSDI

Projet de Modélisation des Données des Systèmes d'Information dans le cadre de l'enseignement en 4IR à l'INSA de Toulouse.

À partir d'un cahier des charges, concevoir une base de données en XML pour la gestion des évênements du LAAS. Générer différentes vues de ces données en HTML/CSS via une transformation XSLT.
